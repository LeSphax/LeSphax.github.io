UX Design Intern at CityMeo


Dates : March 2016 - September 2016 (in progress)
Title : Analysing and improving the usability of a web application

Technologies : HTML, Javascript, CSS, Python
Librairies : Django Framework, JQuery
UX Design : Axure, Google Docs and Sheets, Draw.io, Balsamiq, GIMP
Software : VirtualBox, Debian

A wireframe and a persona I created during the internship

CityMeo is a french startup working on digital signage. It allows companies and people to create, manage and broadcast medias on screens using a web application. It is used, for example, by advertising networks to broadcast advertisement in malls and companies to pass internal messages in offices.

The goal of my internship is to improve the usability of the web application and to make sure it can support scaling up as CityMeo keeps growing. For this I have been conducting interviews with existing users of the platform and I am creating wireframes of a future design.

I have also been working on the existing web application, reporting usability problems and wireframing new functionnalities asked by clients. Furthermore, I am fixing some usability problems and adding small functionalities to the application.

Software Development Intern at the IRIT, ICS Team.


Dates : May 2015 - August 2015
Title : Development of a tablet application platform in an environment of helicopter cockpits

Technologies : Java, Object-Oriented Petri Nets, C
Librairies : JavaFX, Java sockets, X-Plane SDK

This internship taught me a lot about working as part of a professional team :
  • Chosing what to work on based on my different assignments
  • Presenting my work clearly and efficiently, making sure it can be reused
  • Using collaborative software like Maven and Mercurial

This project was about introducing new interactions techniques like touch and haptic feedback in helicopters cockpits. The project was a partnership between Airbus Helicopters, Bertin Technologies and team ICS of the IRIT laboratory.
To confirm the solutions they would find were adapted, the team chose to conduct user tests with pilots.
For the tests they chose to simulate an helicopter cockpit

Architecture of the whole system


Working with another intern, I created a software platform managing communications between the flight simulator and the tablets. The goal of this platform was to allow the developpers of the team to easily prototype and test the solutions they found.
